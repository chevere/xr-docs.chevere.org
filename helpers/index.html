# Helpers

XR Debug provides three helpers to conveniently debug your code:

  • XR Debug xr
  • XR Debug Raw xrr
  • XR Debug Inspector xri

Additionally, the VarDump (opens new window) package provides two extra dump helpers which write to the output stream:

  • VarDump vd
  • VarDump and die vdd
