# Docker configuration

When using Docker (local) the host config should point to the internal IP of your Docker host by using host.docker.internal.

ğŸ’¡ Implement this when needing to debug code emitted by applications running in containers.

# File-based

<?php

return [
    // ...
    'host' => 'host.docker.internal',
];

# Code-based

use Chevere\Xr\Xr;
use Chevere\Xr\XrInstance;

$host = 'host.docker.internal';
new XrInstance(
    new Xr(
        enable: $enable,
        host: $host,
        port: $port
    )
);
